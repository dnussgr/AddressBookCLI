# AddressBookCLI
Ein einfaches Java-Konsolenprogramm zur Verwaltung eines Adressbuchs mit csv-Speicherung.

## Features
- CSV-basierte Datenspeicherung
- Einfaches textbasiertes Menü (CLI)
- Adresssuche nach Vorname, Nachname, PLZ, etc.
- Neue Adressen hinzufügen
- Bestehende Einträge bearbeiten
- Einträge löschen
- Übersichtliche tabellarische Ausgabe

## Technologie
- Java (mind. JDK 11+)
- CLI (Konsolenanwendung)
- Keine externen Bibliotheken

Dieses Projekt wurde während meiner Abschlussprüfung in 6 Stunden erstellt und unverändert hochgeladen.

## Starten

### 1. Kompilieren
```bash
javac Main.java
```
### 2. Starten
```bash
java Main
```
